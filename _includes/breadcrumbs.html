{%- comment -%} Emits a BreadcrumbList JSON-LD block plus a visible breadcrumb nav. Designed to be placed immediately after the hero section (before main content) on pages that use a custom layout rather than _layouts/post.html. Required parameters: section -- middle breadcrumb label (e.g. "Solutions", "Resources") section_url -- URL the middle breadcrumb links to (absolute or root-relative) current -- final breadcrumb label (the current page's title) Usage: {% include breadcrumbs.html section="Solutions" section_url="/#solutions" current="AI Agent Development" %} {%- endcomment -%}
